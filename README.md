# Souffleur.ai

I'm a product manager and drama teacher who wants to help actors memorize their lines.

This project is my attempt to build a tool that actually fits how actors work — learning cues, testing recall, getting gentle prompts when they blank — instead of just reading scripts off a screen.

I'm going to vibe code this entirely in the open. No roadmap, no promises, just building in public and seeing where it goes.

---

**Stack:** Next.js · Supabase · TypeScript
